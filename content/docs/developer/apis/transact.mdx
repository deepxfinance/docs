Submit a signed EVM transaction to chain and parse order id from on-chain events.

Supported `marketType` values:
- `Perp`
- `Spot`

Supported `action` values:
- `PlaceOrder`
- `CancelOrder`
- `ClosePosition`

Supported combinations:
- `Perp + PlaceOrder`
- `Perp + CancelOrder`
- `Perp + ClosePosition`
- `Spot + PlaceOrder`
- `Spot + CancelOrder`

Not supported:
- `Spot + ClosePosition`

Request example:
`POST /v2/chain/tx/transact`
